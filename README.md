# Physical AI & Humanoid Robotics Textbook

This is an educational platform for AI & Robotics students, engineers, and hackathon participants that teaches how to connect digital AI systems with physical robots using industry-standard tools like ROS 2, Gazebo, and NVIDIA Isaac.

## Installation

1. Make sure you have Node.js (v18 or higher) installed
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Available Scripts

- `npm start` - Starts the development server
- `npm build` - Builds the static site for production
- `npm serve` - Serves the built site locally

## Project Structure

The textbook content is organized into 4 modules:
1. Module 1: The Robotic Nervous System (ROS 2)
2. Module 2: The Digital Twin (Gazebo & Unity)
3. Module 3: The AI-Robot Brain (NVIDIA Isaac)
4. Module 4: Vision-Language-Action (VLA)

Each module contains 3 chapters, and each chapter contains 3 lessons, following the Physical AI & Humanoid Robotics constitution.
